# Cryptocurrency Market Dashboard (Power BI)

## 📊 Project Overview

This project is an interactive **Cryptocurrency Market Dashboard** built using **Power BI**.
The dashboard connects directly to an online data source and retrieves updated cryptocurrency market data.

The dataset refreshes automatically when the report is refreshed, allowing the dashboard to display **updated market information every 24 hours**.

The objective of this project is to demonstrate skills in:

* Power BI data connectivity
* Data modeling
* Interactive dashboard creation
* Data visualization and insights

---

## 📁 Project File

* **cryptobi.pbix** – Power BI report file containing the data connection, data model, and visualizations.

---

## 🔗 Data Source

The dashboard is connected to an **online cryptocurrency data source**.
Whenever the report is refreshed in Power BI, the latest data is fetched from the website.

This ensures the dashboard always reflects **recent cryptocurrency market trends**.

---

## 🛠 Tools & Technologies Used

* **Power BI Desktop**
* **Web Data Source Connection**
* **Power Query**
* **DAX (Data Analysis Expressions)**

## 📈 Dashboard Features

### Market Overview

### Price Analysis

* Cryptocurrency price comparisons

### Trading Volume Analysis

* Trading volume by cryptocurrency

### Interactive Filtering

Users can interact with the dashboard using:

* Filters
* Slicers
* Dynamic visuals

This allows users to explore the cryptocurrency market data more effectively.

---

## 📊 Key Visualizations

The report includes multiple Power BI visuals such as:

* KPI Cards
* Bar Charts
* Column Charts
* Tables
* Interactive Slicers

These visuals help users understand market trends and compare cryptocurrencies.

---

## 🔄 Data Refresh

The report retrieves data directly from the website.
When the **Power BI refresh button** is used, the dataset updates with the **latest available cryptocurrency data**.

Data is typically refreshed **every 24 hours** based on the source update.

---

## 🚀 How to Use

1. Download this repository.
2. Open **`cryptobi.pbix`** in **Power BI Desktop**.
3. Click **Refresh** to fetch the latest data.
4. Explore the dashboard using the available filters and visuals.
